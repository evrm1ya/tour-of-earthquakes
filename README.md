# Tour of Earthquakes

## Setup
* `git clone https://github.com/evrm1ya/tour-of-earthquakes.git <project-dir>`
* `cd <project-dir>`
* `npm install`

## Run Dev Server
* `npm start`

## Run Tests (watches automatically)
* `npm run test`
